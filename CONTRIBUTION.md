# Contribution Guidelines

We welcome contributions to this Datacore component!

## Local Development Workflow

1. This component runs locally within the Datacore environment.
2. Ensure you have the `Datacore` plugin active.
3. Make changes in `src/index.jsx`.
4. Reload the note to re-run the component.

## Pull Request Process

1. No emojis in UI — use Obsidian CSS variables and Lucide icons.
2. No personal environment paths or secrets.
3. Update `README.md` if features or structure change.
4. Submit PRs against the `main` branch.
